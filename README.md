# AI员工平台

## 项目简介

本项目是一个AI员工聘用平台的最小可行性产品（MVP），用户可以浏览AI员工列表、查看详情、提交聘用申请以及查询聘用记录。网站设计风格以极简主义为主，结合动态互动元素，提供高效、直观的用户体验。

## 功能特点

- **首页介绍**：展示AI员工形象和平台价值主张
- **AI员工列表**：浏览所有可聘用的AI员工
- **AI员工详情**：查看AI员工的详细技能和工作案例
- **聘用申请**：提交聘用意向表单
- **聘用记录查询**：通过邮箱查询已提交的聘用记录

## 技术栈

- 前端：HTML, CSS (Tailwind CSS), JavaScript
- 动画：GSAP
- 部署：Netlify

## 本地开发

1. 克隆项目
```
git clone [项目地址]
cd ai-employee-platform
```

2. 安装依赖
```
npm install
```

3. 启动开发服务器
```
npm start
```

## 部署

项目配置为通过Netlify自动部署。当代码推送到主分支时，将自动触发构建和部署流程。

### 部署配置

- **构建命令**：`npm run build`
- **发布目录**：`dist`
- **单页应用重定向**：已配置 `_redirects` 文件以支持前端路由

## 项目结构

```
/
├── index.html          # 主HTML文件
├── public/             # 静态资源目录
│   └── _redirects      # Netlify重定向配置
├── package.json        # 项目配置和依赖
└── README.md           # 项目说明文档
```
